## Internals

Beim Start eines Boards wird gepr√ºft ob die SSID (EEPROM) bereits vorhanden ist.

* Wenn ja, wird das Board an diese SSID als Client verbunden
* Wenn nein, wird ein AccessPoint (AP) mit dieser SSID gestartet

Sollte das "Master" Board nicht mehr verf√ºgbar sein (z.B. Stromausfall), kann durch neustart eines beliebigen Boards ein neuer "Master" definiert werden. Die anderen Boards verbinden sich automatisch an diesen als "Client".

